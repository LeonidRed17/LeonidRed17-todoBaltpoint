# TodoList

Проект TodoList на Laravel 11 + PostgreSQL. Управление группами задач и задачами.

## Установка и запуск

### git clone https://github.com/LeonidRed17/todoBaltpoint.git
### cd todoBaltpoint
### composer install
### npm install
### Переименовать .env.example в .env
### php artisan migrate:fresh --seed
### composer dev
